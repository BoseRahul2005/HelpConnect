# MadadSetu 🌱

MadadSetu is a community-driven NGO platform that connects people with trusted NGOs, social causes, and donation opportunities. The platform is designed to make helping easier, faster, and more transparent by bringing communities and organizations together in one place.

## ✨ Features

- 🏠 Modern homepage with clean NGO-focused UI
- 🔍 Search NGOs, causes, or posts
- 📂 Category-based NGO filtering
- 📢 Feed section for NGO campaigns and updates
- ❤️ Donate to verified causes
- 📈 Popular NGOs section
- 🚨 Urgent cases highlight section
- 👤 User profile & notifications
- 📱 Responsive design for desktop and mobile

## 🎯 Categories

- Education
- Healthcare
- Disaster Relief
- Animal Welfare
- More coming soon...

## 🚀 Mission

To create a digital bridge between people who want to help and organizations that need support.
